# 描述文件

# id

  • 类型: string

插件标识符，算法容器、插件数据库，接口都会基于这个ID进行命名

# name

  • 类型: string

插件名称，前端展示

# version

  • 类型: string

插件版本号，根据版本号进行升级操作

# description

  • 类型: string

插件详细描述，前端展示

# use

  • 类型: number

插件算法占用的算力 单位TOPS

# defaultConfig

  • 类型: object

算法默认配置信息，没有的话可以留空

# group

  • 类型: array
  • 示例: ["tdds", "gzyssb"]

算法作为组件发布时，需要group属性，内容为组件包含的插件ID

# color

  • 类型: string
  • 示例: #412680

插件的颜色，插件在算法管理页面展示时的主题颜色

# background

  • 类型: string
  • 示例: linear-gradient(219deg,#d0bcff 0%, #412680 94%)

插件的颜色，插件在算法管理页面展示时的背景
